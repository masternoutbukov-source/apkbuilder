# МеталлСмета 48

Flutter-проект для расчета:

- забора из профнастила
- гаража из профтрубы и профнастила
- листового металла
- общей сметы
- карт распила трубы
- резов под 45°
- раскроя профнастила по секциям, воротам и калитке
- сохранения проекта в JSON
- экспорта сметы в PDF

## Что внутри

- `lib/main.dart` — весь код MVP в одном файле
- `pubspec.yaml` — зависимости проекта
- сохранение в `project_latest.json` и архивные JSON-файлы в папке приложения
- экспорт `estimate_YYYYMMDD_HHMMSS.pdf` в папку приложения

## Как запустить

1. Установи Flutter SDK и Android toolchain по официальной инструкции Flutter.
2. Открой папку проекта в терминале.
3. Если в папке нет стандартных Android-файлов Flutter, сгенерируй их:

```bash
flutter create --platforms=android .
```

4. Получи зависимости:

```bash
flutter pub get
```

5. Запусти на устройстве:

```bash
flutter run
```

6. Собери APK:

```bash
flutter build apk
```

Готовый release APK обычно появится по пути:

```text
build/app/outputs/flutter-apk/app-release.apk
```

## Что добавлено в этой версии

- точные полосы профнастила по полезной ширине
- отдельный раскрой для ворот и калитки
- карта распила трубы по хлыстам
- учет пропила пилы
- резы под 45° для рам ворот и калиток
- сохранение и восстановление проекта через `project_latest.json`
- экспорт итоговой сметы в PDF

## Примечания

- PDF генерируется без внешних шрифтов, поэтому шаблон сделан в ASCII-safe формате.
- Цены поставщиков встроены как стартовый каталог и могут быть изменены вручную.
- Формулы расчета — инженерный MVP для первичной сметы. Перед закупкой сверяй спецификацию, толщины, узлы крепления и реальный раскрой.
